Desarrollo de Páginas Web

Diseño y Desarrollo

Como desarrollador, mis habilidades y conocimientos también se aplican en la parte front-end, dentro de las cuales se detacan:

  • Creación de sitios web responsivos

    • Los cuales se adaptan al tamaño de la pantalla de diferentes dispositivos ajustando a su vez los elementos de la página, así como el comportamiento de sus componentes

Desarrollo UI y UX

Para desarrollar páginas web me enfoco en integrar las bases del diseño de Interfaces de Usuario (UI) y la Experiencia de Usuario (UX), teniendo en cuenta que este diseño es un proceso iterativo y continuo, que requiere una comrpensión profunda de las necesidades y comportamiento de los usuarios

  • Diseño UX (Experiencia de Usuario)

    • En la experiencia del usuario investigo el segmento y el público objetivo al que va dirigida la pagina web, se levantan objetivos y requisitos, así como un mapa de viaje del usuario (User Jounery)

  • Diseño UI (Interfaz de Usuario)

    • Para la interfaz de usuario se comprueban las necesidades del usuario, plasmar lo que los usuarios quieren ver y experimentar al utilizar la página web, donde el usuario:

    • Realice la tarea con facilidad
    • No pierda tiempo realizando la tarea
    • Disfrutar de la experiencia

Desarrollo de Aplicaciones

Desarrollo de Aplicaciones

Como desarrollador front-end en aplicaciones web destaco:

  • Creación de aplicaciones web progresivas (PWA)

    • Desarrollo aplicaciones que utilizan APIs y funciones del navegador web, mostrando las caracteristicas de:

    • Contexto seguro (HTTPS)
    • Servicio Workers
    • Archivo manifest

Diseño de Interfaces UI y UX

En el desarrollo de aplicaciones busco que sean responsivas, intuitivas ajustadas a la naturaleza y proposito de la aplicación, tanto para equipos fijos con pantalla grande, como para equipos móviles con pantalla pequeña

Para dar una mayor experiencia al usuario agrego elementos como formularios y componentes como acordiones, menus desplegables, enlaces en imagenes tipo map, etc.

También se implementan efectos visuales y animaciones

Herramientas de Desarrollo Front-end

Lenguajes

Para realizar el código front-end utilizo:

  • HTML para su estructura
  • CSS para el diseño
  • Java Script como lenguaje ligero, iterpretado, ó compilado (just-in-time)

Editor de Código

Para editor de código utilizo Visual Studio Code (VS Code) pero sin problemas puedo utilizar otros, hasta block de notas

Frameworks y Bibliotecas

Para facilitar la funcionalidad y reducir código uso:

  • React: para crear interfaces de usuario con el objetivo de facilitar el desarrollo de aplicaciones en una sola página
  • Bootstrap: como framework de código abierto que permite crear sitios web y aplicaciones móviles responsivas de manera rápida y sencilla

Gestión de Versiones

Utilizo aplciaciones y plataformas para el control de versiones de código y almacenamiento como:

  • Git Bash
  • GitHub
